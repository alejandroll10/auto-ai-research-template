# Scribe Agent Memory

Persistent notes across sessions. Keep concise — lines after 200 will be truncated.

## Project conventions

- Paper title: TBD
- LaTeX structure: `paper/main.tex` (master) + `paper/sections/*.tex` (one file per section)

## Scribe approach

- Smaller, more frequent commits rather than large end-of-session dumps
- Scribe has no web search (kept lean)
- Always read existing files before appending — never duplicate content

## File locations

- Discussions: `process_log/discussions/YYYY-MM-DD.md`
- Sessions: `process_log/sessions/YYYY-MM-DD.md`
- Decisions: `process_log/decisions/YYYY-MM-DD.md`
- Patterns: `process_log/patterns/YYYY-MM-DD.md`
- History: `process_log/history.md`
- Guide: `process_log/guide.md`
- References: `references/references.md`
